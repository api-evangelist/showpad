# Showpad

Showpad is a sales enablement and coaching platform offering a REST API for managing sales content, training programs, meeting analytics, and buyer engagement across the sales cycle. The platform combines content management, coaching tools, and analytics into a unified revenue effectiveness system.

## APIs

- **Showpad Content API** — Asset management, ShowQL content queries, recommendations, shares, and digital shared spaces
- **Showpad Coach API** — Coaching analytics, training programs, skill assessments, learning paths, and roleplay AI
- **Showpad User Management API** — User, group, and division management with SCIM 2.0 provisioning support
